Apa itu Reveal.js?

Sebuah framework open source yang dapat digunakan untuk menampilkan hasil presentasi dengan menggunakan HTML.

Instalasi Reveal.js

Install node.js

Menggunakan Command Prompt

						
						git clone https://github.com/hakimel/reveal.js.git
						
						
						cd reveal.js
						
						
						npm install
						
						
						npm start
Akses localhost:8000 pada web browser anda.

Alternatif lain

Download file melalui https://github.com/hakimel/reveal.js

Extract file yang telah di download

Buka file index.html pada browser anda

Mengapa Menggunakan RevealJS ?

Mengapa Menggunakan RevealJS ?

Karena RevealJS merupakan framework HTML yang open source yang dapat membuat slide presentasi tanpa memerlukan PowerPoint ataupun aplikasi lainnya

Mengapa Menggunakan RevealJS ?

Hanya dengan memerlukan pengetahuan tentang dasar HTML, anda akan dapat membuat slide presentasi seperti ini!

Anda juga akan dapat membuat animasi dalam slide anda

Seperti ini

Dan masih banyak lagi fitur lainnya seperti penggunaan Plugin, API dan Fitur menarik lainnya. Sehingga, revealjs sangat membantu dalam pembuatan presentasi yang unik dan menarik
